# 🛠️ JESHA POS

Un sistema de punto de venta (POS) integral y escalable diseñado específicamente para optimizar la gestión y operativa diaria de negocios en el sector ferretero.

## ✨ Características Principales

* **Gestión de Inventario:** Control preciso de existencias, facilitando el seguimiento de entradas, salidas y stock de productos ferreteros.
* **Soporte Multi-sucursal:** Administración centralizada que permite operar y monitorear múltiples ubicaciones desde una sola plataforma.
* **Integración Fiscal:** Módulo dedicado para la gestión de facturación y cumplimiento de los requerimientos fiscales correspondientes.
* **Agilidad en Mostrador:** Interfaz optimizada para transacciones rápidas y seguras con los clientes.

## 💻 Tecnologías Utilizadas

El sistema está construido sobre un stack moderno y eficiente para garantizar escalabilidad y alto rendimiento:

* **Frontend:** React / React Native (para soluciones móviles).
* **Backend:** Node.js con Express y microservicios/scripts en Python.
* **Base de Datos & ORM:** Prisma.
